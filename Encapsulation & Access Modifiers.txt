Encapsulation and Access Modifiers
-----------------------------------------------
In most cases , data should not be directly accessible outside of the class . This concept of hiding implementation is known as encapsulation and to achieve Encapsulation we use Access Modifiers .

Access Modifiers
----------------------
1. public 
    - Visibility : Everywhere 
2. private
    - Visibility : Only within its own class
3. default
    - Visibility : Only within the package. 
4. protected
    - Visibility : Within the package and outside the package through child class.
